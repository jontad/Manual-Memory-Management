Clona git repot

När du pullar så skriver du

git pull origin master

Då får du senaste koden i master

För att byta branch skriver du

git checkout <branch> utan <> med branch = din branch

Nu kan du göra alla ändringar du vill, adda allt och commita med bra beskrivning

Sen kan du pusha upp allt med

git push origin <branch> utan <> där branch = din branch

Sen kan du gå in på github och starta en ny pull-request.

Ha base: Master och Compare: <din branch>

Ha en titel och skriv i description vad du har ändrat på.

Låt någon annan gå igenom och acceptera, helst 2 personer.
